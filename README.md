# rivvl-v2

AI-powered comparison platform — Vehicles and Real Estate verticals.
Built with Next.js 14, TypeScript, Tailwind CSS, Supabase, Stripe, Firecrawl, and the Anthropic Claude API.

## Setup

1. Copy `.env.local.example` to `.env.local` and fill in all API keys
2. Run: `npm install`
3. Run: `npm run dev`

See the developer handoff document for full setup instructions, environment variables, and deployment guide.
